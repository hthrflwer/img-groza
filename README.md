# Хранилище картинок Грозы

Для того, чтобы Вы могли добавлять изображения в репозиторий обратитесь [сюда](https://vk.com/cassiope_a), чтобы вас добавили в качестве коллаборатора, вам нужен аккаунт на GitHub для этого. 

## Инструкция для не программистов о добавлении изображений в репозиторий

### 1. Добавление картинки в папку

1. Откройте нужную папку, кликнув по названию.
2. Нажмите зелёную кнопку **"Add file"** (справа сверху над списком файлов).
3. В выпадающем меню выберите **"Upload files"**.
4. Перетащите картинку в открывшуюся область страницы, либо нажмите **"choose your files"** и выберите файл через проводник.
5. Внизу страницы есть поле **"Commit changes"** — впишите туда короткое пояснение, например:
   `Добавлен фон для зарослей орешника`
6. Нажмите зелёную кнопку **"Commit changes"**.

Готово!

Для добавления новых папок в репозиторий обратитесь [сюда](https://vk.com/cassiope_a).

### 2. Замена уже существующей картинки
 
Если нужно обновить картинку, оставив то же имя файла:
 
1. Повторите шаги 1–5 из раздела выше, но выберите файл **с точно таким же именем**, как у старого (например, `logo.png` вместо `logo.png`).
2. GitHub сам заметит совпадение и покажет пометку, что файл будет заменён.
3. Впишите комментарий, например: `Обновлена картинка logo.png`.
4. Нажмите **"Commit changes"**.

### 3. Удаление ненужной картинки
 
1. Откройте файл, кликнув на его имя в списке репозитория.
2. Найдите иконку корзины (обычно справа сверху над содержимым файла).
3. Нажмите на неё.
4. Внизу впишите комментарий и нажмите **"Commit changes"**.
